# Archived Tests

Эта папка содержит тесты из ранних версий проекта (POC фаза), которые использовали старый API до рефакторинга по SOLID принципам.

## Файлы

- `test_search.py.bak` - тесты поиска из оригинального POC
- `test_splitter.py.bak` - тесты сплиттера из оригинального POC

## Зачем сохранены?

1. **Историческая справка** - показывают эволюцию проекта
2. **Reference** - можно подсмотреть edge cases и сценарии использования
3. **Документация** - примеры того, как работал старый API

## Статус

⚠️ Эти тесты **НЕ запускаются** автоматически (папка с префиксом `_`).

Они могут не работать с текущей архитектурой, так как API был значительно изменен после Phase 1 рефакторинга.

## Актуальные тесты

Смотрите корневую папку `tests/`:

- `test_phase_1_architecture.py` - проверка контрактов и DTO
- `test_phase_2_storage.py` - проверка Peewee adapter, RRF, фильтров
- (будут добавлены после Phase 3) - integration тесты через SemanticIndex
