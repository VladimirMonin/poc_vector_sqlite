О проекте

Semantic Core — это библиотека для создания интеллектуальных систем поиска и анализа документов. Основана на технологиях:

  • SQLite + sqlite-vec — локальное хранение векторов без внешних зависимостей
  • Google Gemini — генерация эмбеддингов и мультимодальный анализ
  • Hybrid Search (RRF) — комбинация векторного и полнотекстового поиска
  • RAG — ответы на вопросы на основе базы знаний
Возможности
  • Семантический поиск
  • Гибридный поиск (RRF)
  • RAG чат с источниками
  • Markdown парсинг
  • Анализ изображений
  • Транскрипция аудио
  • Видео анализ
  • Batch API (50% экономия)
Технологический стек
Backend Flask 3
Frontend HTMX

Database SQLite
ORM Peewee
AI Gemini
Vectors sqlite-vec
Config Pydantic
